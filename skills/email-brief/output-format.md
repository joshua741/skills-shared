# Output Format — Email Brief Skill

## Placement in Notion

Write to the `EMAIL BRIEFING` section of the Notion daily page. Position directly below TODAY'S MEETINGS. Morning and afternoon briefings each have their own labeled block. Do not overwrite the prior run's briefing — append or replace only the current run's labeled block.

---

## Format Rules

**Summary field:** Always specific. Include the relevant number, date, address, name, or dollar amount. Never write "someone sent an email about a property" — write "Mike Torres confirmed the $287,500 purchase price on 4412 Birchwood and is waiting on the PSA."

**Task match field:** Use the exact task name as written in Notion. Do not paraphrase. If there is no match, write "No direct task match" — do not invent a connection.

**Pushes needle:** Answer Yes or No. If Yes, state the specific reason in one clause. "Yes — title has the payoff figure Joshua needs to close the funding gap."

**Reply draft:** Full draft, ready to send with no editing needed. Do not write "[draft]" or "[see reply-style.md]" — write the actual draft. See `reply-style.md`.

**New task:** Only write tasks where the implication is clear from the email. Do not manufacture tasks. When in doubt, skip and note in the summary instead.

**Secondary emails:** One line only. If it takes more than one line to explain, it might be Tier 1.

**Auto-handled:** Sender names only, comma-separated. Joshua does not need to see what these emails said.

---

## Priority Ordering Within Each Section

Priority Emails order:
1. Needle Mover matches (these also appear in the dedicated section above)
2. Emails with time-sensitive content (deadlines, same-day decisions, payments)
3. Emails with direct task matches
4. Remaining Tier 1 emails by recency

Secondary emails: chronological, most recent first.
